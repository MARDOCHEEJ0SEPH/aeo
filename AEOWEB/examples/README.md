## Installation (General)

### Static Site (Any Example)

```bash
# 1. Navigate to example
cd examples/[example-name]

# 2. Customize
# - Edit schema/*.json files
# - Update index.html
# - Replace images

# 3. Deploy
netlify deploy --dir=. --prod
```

### Full-Stack (Any Example)

```bash
# 1. Database setup
createdb [your_db_name]
psql -d [your_db_name] -f database/schema.sql
psql -d [your_db_name] -f database/seed.sql

# 2. Backend setup
cd backend
npm install
cp .env.example .env
# Edit .env
npm run dev

# 3. Test
curl http://localhost:3000/api/health
```
