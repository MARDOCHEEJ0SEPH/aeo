## ğŸš€ Quick Start Guide

### Prerequisites

- Docker & Docker Compose installed
- OpenAI API key
- Anthropic API key (optional but recommended)
- 8GB+ RAM recommended
- 20GB+ disk space

### Installation

1. **Clone the repository**
```bash
cd autonomous-robotics
```

2. **Create environment file**
```bash
cp .env.example .env
# Edit .env and add your API keys
```

3. **Start the system**
```bash
docker-compose up -d
```

4. **Check system health**
```bash
curl http://localhost:3000/health
```

5. **Access the platform**
- Frontend: http://localhost:8080
- Neural Core: http://localhost:3000
- Analytics: http://localhost:3001
- API Gateway: http://localhost:3000

### First Time Setup

The system will automatically:
1. Initialize database schemas
2. Create Elasticsearch indices
3. Set up Redis cache
4. Train initial neural networks
5. Start evolution cycles
6. Begin content generation

**Note**: Full initialization takes 5-10 minutes on first run.
